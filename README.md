# EPUB 导出器（SillyTavern 插件）

把当前角色的聊天记录一键导出为 **EPUB 电子书**——**所见即所得**：你在酒馆里看到什么文字和图片，导出的就是什么，图片保持在原文位置。

## 特性

- 📚 导出为标准 EPUB，手机/阅读器/Kindle 都能看
- 👁️ **所见即所得**：读取页面上渲染后的可见内容（正则切除、markdown 渲染之后），不是原始存储文本
- 🖼️ 聊天里的图片自动打包进电子书，位置和你看到的一致
- ✅ 可选导出的消息类型（用户 / 助手 / 系统）
- 🎯 右下角悬浮球，可拖动，点开即用

## 安装

在 SillyTavern 里：**扩展（Extensions）→ Install extension**，粘贴本仓库地址：

```
https://github.com/euphoriaaaaaa1/sillyconverter-plugin
```

回车即可。装好后扩展列表会出现「EPUB 导出器」，页面右下角出现 📚 悬浮球。

> 需要能访问 `cdnjs.cloudflare.com`（运行时加载 JSZip 打包库）。

## 使用

1. 打开一个聊天
2. 点右下角 📚 悬浮球，弹出面板
3. 勾选要导出的消息类型 → 点「导出为 EPUB」
4. 浏览器自动下载 `.epub` 文件

## 已知边界

- 只导出消息正文里的文字和**内联图片**；作为「附件图」单独挂在正文外的图片暂不包含。
- 依赖 SillyTavern 的消息 DOM 结构（`.mes` / `.mes_text`），如遇 ST 大版本改动可能需要适配。
